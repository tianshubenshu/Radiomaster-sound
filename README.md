本项目使用GPT-Sovits制作，利用游戏wiki音频进行训练。  
项目地址：https://github.com/RVC-Boss/GPT-SoVITS  
部分语音数据及模型来源于开源分享，转载及使用请标注其作者。 


理论适用于所有EDGETX系统遥控器。  
将对应语音文件夹拖入遥控器sd卡并选择覆盖即可，具体使用方法可参考b站教程或EDGETX wiki
